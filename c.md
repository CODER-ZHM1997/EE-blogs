## pointer

指针变量为什么能够被赋值？

- 因为它存的是地址

char* 是字符串指针，其实也可以叫数组指针

int* 是（整数）数组指针

指针也可以指向一段空白的数组

```c
char *a = malloc(10 * sizeof(char));
```



## array

数组变量与数组指针不同

- 数组变量是常量指针，不能修改，值是确定的，而且数组大小是确定的，
- 数组指针是普通指针，不能确定数组大小

## typeof
